# รายงานสรุปผลการทดลองปรับปรุง Prompt V3: กลุ่ม Flat Classification (Exp 1E)

รายงานฉบับนี้สรุปผลลัพธ์ของ **Experiment 1E (Flat Zero-Shot Classification)** เปรียบเทียบระหว่าง:
*   **Prompt V2 (Optimized Prompt เดิม)**: ปลดล็อกกฎเกณฑ์ตึงตัวดั้งเดิม
*   **Prompt V3 (New Prompt ปรับปรุงล่าสุด)**: เพิ่มลำดับการตัดสินใจ (Critical Decision Hierarchy) และกฎการแยกแยะเคสสับสน (Edge-Case Resolution Rules)

## 2. การวิเคราะห์ผลลัพธ์ (V3 vs. V2)

### 📈 สิ่งที่พัฒนาขึ้น (Key Improvements)
1.  **Gemma-4 มี Category F1 สูงสุดขึ้นอย่างเด่นชัด**:
    *   ที่อุณหภูมิต่ำสุด (Temp 0.0) Gemma-4 มีคะแนน Category F1 ดีขึ้นอย่างก้าวกระโดดจาก `0.6234` เป็น **`0.6394` (+1.60%)** ซึ่งเป็นผลดีโดยตรงจากการใช้ **Hierarchy Order** ที่เข้ามาช่วยโมเดลคัดแยกระหว่างผู้ประสบภัย (Displacement) และความเสียหายทางวัตถุ
2.  **DeepSeek-v4-flash ทำผลงานคงเส้นคงวาและดีขึ้นในบาง Temp**:
    *   ที่ Temp 0.1 คะแนน Category F1 ดีขึ้น **+1.81%** (ขยับจาก `0.5915` เป็น `0.6096`) และที่ Temp 0.3 ดีขึ้น **+0.97%**
    *   แสดงให้เห็นว่า DeepSeek ตอบสนองเชิงบวกกับกฎ Edge-case resolution ที่ขยายเพิ่มใน V3

### 📉 จุดอ่อนหรือข้อควรระวัง (Performance Drops)
1.  **Typhoon-v2.5 ได้รับผลกระทบเชิงลบ**:
    *   คะแนน Category F1 ของ Typhoon ร่วงลงเฉลี่ย **-2.1% ถึง -3.6%** ในทุกระดับอุณหภูมิ (ร่วงต่ำกว่าเกณฑ์ `0.60` ลงมาอยู่ที่ `~0.58 - 0.59`)
    *   *เหตุผลที่เป็นไปได้:* การเพิ่มกฎและ Hierarchy ภาษาอังกฤษที่ยาวและซับซ้อนขึ้น อาจสร้างความสับสนหรือมีคำรบกวน (Context Clutter) มากเกินไปสำหรับ Typhoon ซึ่งมีพารามิเตอร์และการแปลความภาษาอังกฤษที่จำกัดกว่าโมเดลขนาดใหญ่อย่าง DeepSeek และ Gemma

## 3. ข้อเสนอแนะเชิงกลยุทธ์ (Recommendations)

*   หากระบบเลือกใช้โมเดล **Gemma-4** หรือ **DeepSeek-v4-flash**: แนะนำให้สลับมาใช้ **Prompt V3** ที่ Temp 0.0 หรือ 0.1 เนื่องจากให้ความเที่ยงตรงในการคัดแยกประเภท (Category F1) สูงกว่า
*   หากระบบเลือกใช้โมเดล **Typhoon-v2.5**: แนะนำให้ใช้ **Prompt V2** ตัวเดิมจะคุ้มค่าและได้ความถูกต้อง F1 สูงที่สุด (`0.6201` ที่ Temp 0.3)
