# รายงานสรุปผลการทดลองปรับปรุง Prompt V3: กลุ่ม Two-Layer Joint Classification (Exp 2E)

รายงานฉบับนี้สรุปผลลัพธ์ของ **Experiment 2E (Two-Layer Joint Classification)** เปรียบเทียบระหว่าง:
*   **Prompt V2 (Optimized Prompt เดิม)**: ปลดล็อกกฎเกณฑ์ตึงตัวดั้งเดิมในแบบสองเลเยอร์พร้อมกัน
*   **Prompt V3 (New Prompt ปรับปรุงล่าสุด)**: เพิ่มความเข้มงวดในตรรกะความสอดคล้อง (Consistency Rule), กำหนดลำดับการตัดสินใจ (Critical Decision Hierarchy) และ Edge-Case Rules

## 2. การวิเคราะห์ผลลัพธ์ (V3 vs. V2)

### 📈 สิ่งที่พัฒนาขึ้น (Key Improvements)
1.  **DeepSeek-v4-flash พัฒนาขึ้นอย่างชัดเจนในทุกช่วงอุณหภูมิ**:
    *   ได้รับ F1-Score สูงขึ้นทั้งด้าน Informativeness และ Category 
    *   โดยเฉพาะอย่างยิ่ง **Category F1 ที่อุณหภูมิ 0.2 ขยับขึ้นจาก `0.6018` เป็น `0.6224` (+2.06%)**
2.  **Gemma-4 เติบโตอย่างก้าวกระโดด**:
    *   Category F1 สูงขึ้นในทุกจุดของการทดสอบ โดยมีคะแนนสูงสุดที่ Temp 0.2 เพิ่มขึ้นจาก `0.5967` มาอยู่ที่ **`0.6139` (+1.72%)** และที่ Temp 0.0 แตะระดับ **`0.6150` (+1.14%)**
    *   *ข้อสรุป:* สถาปัตยกรรมแบบสองเลเยอร์ร่วมกัน (Two-Layer Joint) มีความสอดคล้องด้านตรรกะดีขึ้นเมื่อมี Prompt V3 คอยประคับประคอง

### 📉 จุดอ่อนหรือข้อควรระวัง (Performance Drops)
1.  **Typhoon-v2.5 ตกลงไปเล็กน้อยถึงปานกลาง**:
    *   คะแนน Category F1 ของ Typhoon ตกลงประมาณ **-0.7% ถึง -3.0%**
    *   *เหตุผลที่เป็นไปได้:* โครงร่างเงื่อนไขของ Layer 1 และ Layer 2 รวมกันส่งผลให้ความยาว Prompt ของ V3 มีความยาวค่อนข้างสูง ส่งผลให้ Typhoon สับสนในตรรกะการเรียกใช้ API เครื่องมือในบางจุด
